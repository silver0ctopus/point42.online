# Вход

Та самая заглавная страница index.html. 3D-Локация в виде небольшого помещения кабины лифта. При открытии этой страницы в самом начале можно добавить еффекты прибытия лифта на этаж (звуки и покачивание камеры). Справа от дверей лифта кнопка, вызывающая всплывающее окно с небольшим приветствием, где можно авторизоваться (ввести имя и пароль) или выбрать гостевой пропуск. После завершения действий в этом окне, окно закрывается, запускается короткий визуальный эффект открытия дверей и загружается страница “Лобби”
