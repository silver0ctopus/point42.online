# Своя квартира

Небольшая 3D-локация, содержащая следующие объекты: 

дверь наружу (ведёт в “жилой отсек); 

кровать - объект, восстанавливающий здоровье за время; 

телевизор, доступный к просмотру во время отдыха в кровати; 

сундук - место хранения предметов игрока;

стеллаж - место для хранения некоторых предметов;

пустое место для холодильника, где пока ничего нет, но нужно зарезервировать при планировании;

стол - место для некоторых видов крафта.
