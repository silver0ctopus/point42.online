# Технические помещения

Помещение среднего размера. Справа от входа расположена будка скупщика вторсырья с NPC скупщика. Прямо расположены ворота на свалку, слева дверь в подвал. За спиной дверь обратно в лобби. Для прохода и на свалку, и в подвал нужен пропуск прошедшего инструктаж у боевого инструктора.
