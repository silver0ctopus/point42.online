# Свалка

Локация для добычи ресурсов. Возможные события и вероятность их появления:

добыча ресурсов:

доска - 9%;

палка - 9%;

тряпка - 9%;

бутылка - 9%;

кусок провода - 9%;

острая железяка - 9%;

тяжёлая железяка - 9%;

кусок резины - 9%;

почти хорошая еда - 9%;

алюминиевая банка - 9%;

икринка покемона - 5%.

Также при добыче возможно нападение крысы - 5%.
