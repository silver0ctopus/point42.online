# Лобби

3D-локация в виде просторного восьмиугольного зала. В самом центре находится стойка “Информация и Регистрация” - обычная круглая стойка, как в аэропортах или торговых центрах, за которой стоят две фигуры NPC  (Мира и Айра). При нажатии на стойку открывается страница “стойка регистрации”

За спиной игрока находится дверь того самого лифта, нажав на которую всплывает окно с предложением покинуть игру. Будем считать эту стену с дверьми лифта стеной №1. Далее именуем стены простым списком по часовой стрелке. 

Стена 2: Два объекта.

Телевизор. Это стенд, показывающий по умолчанию текст Манифеста. При нажатии открывается страница “Телевизор”, доступная для всех.

Стенд с газетами. При нажатии открывается страница “газетный стенд” доступная для всех, на которой можно почитать свежий выпуск официальной газеты прямо на стенде или купить этот или один из других выпусков официальной газеты или других изданий. Официальная газета бесплатная, другие издания распространяются на усмотрение их редакций. Покупка доступна только авторизованным пользователям. При покупке предмет “газета” появится в инвентаре.

Стена 3: Вход в жилые помещения. Нужен пропуск сигната. При нажатии гостем появляется всплывающее окно, предлагающее сначала пройти регистрацию. Авторизованных пользователей сразу впускает в локацию “жилой отсек”.

Стена 4: Вход в административные помещения. Нужен пропуск сигната. При нажатии гостем появляется всплывающее окно, предлагающее сначала пройти регистрацию. Авторизованных пользователей сразу впускает в локацию “административные помещения”.

Стена 5: Ворота во внешний мир, пока при нажатии появляется только всплавающее окно с объявлением о том, что пока идут работы по терраформированию и выход недоступен.

Стена 6: Вход в молл. Нужен пропуск сигната. При нажатии гостем появляется всплывающее окно, предлагающее сначала пройти регистрацию. Авторизованных пользователей сразу впускает в локацию “молл”.

Стена 7: Вход в технические помещения. Нужен пропуск сигната. При нажатии гостем появляется всплывающее окно, предлагающее сначала пройти регистрацию. Авторизованных пользователей сразу впускает в локацию “технические помещения”.

Стена 8: Банкомат. Объект в виде банкомата. Пока при нажатии появляется только всплавающее окно с объявлением о том, что банкомат ещё не работает. Позже будет переводить на страницу “банкомат”, где можно будет купить векторы и обменять векторы на скаляры.
