# Директива №4 о Верификации

Верификация сигната - это добровольная процедура, в ходе которой устанавливается уникальность личности сигната и подтверждающая владение только одним комплектом документов. Верификация проводится службой безопасности "Точки 42" по личному заявлению. Верификация не является обязательной для всех сигнатов и служит лишь для предотвращения махинаций в законодательном либо финансовом процессе. Информация, полученная службой безопасности в ходе проведения верификации, не подлежит разглашению и защищена законодательством Точки 42.
