# Директива №3 о Валюте

На территории "Точки 42" существуют два вида валют. Скаляр для всех внутренних расчётов, и вектор для транзакций с экономикой Терры. На период действия директивы №1 свободное хождение вектора не разрешено и он может быть конвертирован в скаляры только в госбанке по официальному курсу. Нелегальная торговля валютами "Точки 42" за деньги Терры категорически запрещена и карается лишением гражданства. Начало рыночной конвертации скаляра и вектора возможно только после отмены директивы №1 и подготовки соответствующей законодательной базы.
