# Директива №5 о Временном Совете

Во время действия стабилизационного периода мандат сенатора Временного Совета может получить каждый сигнат, внесший определённый вклад в развитие проекта. Для определения меры вклада сигната вводятся временные талоны - маркеры. Мандат может получить сигнат, собравший 100 маркеров. Виды работ и размер вознаграждения в маркерах указан в Приложении 1 к данной директиве. Временный Совет не имеет ограничений по количеству сенаторов. Решения принимаются простым большинством.
